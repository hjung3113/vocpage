## 세션 시작 절차

1. `claude-progress.txt` (첫 30줄)
2. `docs/specs/plans/next-session-tasks.md`
3. 필요한 spec만 선택적으로
4. 룰은 `CLAUDE.md` 참조

## 룰

전부 `CLAUDE.md` 가 정본 — 본 파일에는 룰을 복제하지 않는다. Codex/다른 agent 도 `CLAUDE.md` 를 그대로 따른다.
