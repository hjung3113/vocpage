## Design Context

### Users
사내 데이터 분석가, 매니저, 관리자 — 기존 분석 시스템(iframe 임베드) 안에서 VOC를 관리하는 B2B 내부 도구 사용자. 사무실 환경에서 데스크톱 위주로 사용하며, 야간 작업도 잦음. 시스템 테마 설정을 따르는 것이 자연스럽다.

### Brand Personality
**정밀하고, 신뢰할 수 있고, 체계적인** — 삼성 Galaxy/OneUI 디자인 언어에서 영감을 얻은 corporate blue 팔레트. 과하지 않고 군더더기 없이 기능을 전달하는 enterprise 도구의 미감.

### Aesthetic Direction
- **Primary brand color**: Samsung Blue 계열 (`#1428A0` 딥 네이비 베이스)
- **Theme**: `prefers-color-scheme` 시스템 설정 자동 연동 (`color-scheme: light dark`, `light-dark()` CSS 함수 사용)
- **Dark mode**: 딥 블루-블랙 배경 + Samsung Blue 악센트
- **Light mode**: 깨끗한 화이트-블루 코퍼레이트 삼성 앱 느낌
- **Font**: Pretendard Variable (한국어 친화적, 기하학적, 기업 느낌 — 삼성 One UI 유사 타입페이스)
- **Anti-reference**: 보라/인디고 계열, glassmorphism, 네온 글로우 없음

### Design Principles
1. 시스템 테마를 존중한다 — light/dark 모두 동등하게 아름다울 것
2. 삼성 블루가 유일한 크로마틱 색상 — 나머지는 블루-tinted 뉴트럴
3. 기능이 먼저, 장식은 나중 — 정보 밀도 높은 테이블 UI가 핵심
4. OKLCH 팔레트 — 지각적으로 균일한 색상 계층 보장
5. Pretendard Variable weight 로 한국어와 영문 동시에 아름답게
