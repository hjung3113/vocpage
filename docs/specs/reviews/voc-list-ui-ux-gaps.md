## 일치하는 항목 ✅

- VOC 제목 font-size / font-weight / color
- 담당자 아바타 (이니셜 원 + 이름) 스타일
- 등록일 포맷 (YYYY.MM.DD) 및 색상
- 상태 배지 전체 스타일 (color, shape, dot)
- 우선순위 배지 스타일 (↑/^/—/v + 텍스트)
- Chevron 토글 동작 (subtask 있는 row만 활성)
- 서브 VOC 인라인 행 동작 (들여쓰기 + 연한 배경)
- Topbar 레이아웃 (제목 + 카운트 + 검색 + 벨 + 생성 버튼)
- 상태 탭 필터 (전체/접수/검토중/처리중/드랍/완료 + 아이콘)
- 필터 더보기 버튼 위치 (우측 끝)
